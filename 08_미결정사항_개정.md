# PinLog 미결정 사항 — 개정본

아래 항목은 확정된 것처럼 표현하지 않는다.

## AI·검색

- GraphRAG 사용 여부
- Java AI 기반 일반 RAG 사용 여부
- 그래프 관계를 검색에 포함할지
- 임베딩 모델
- Place·Context·Keyword 점수 통합 방식
- 임베딩 생성 동기·비동기 처리
- 임베딩 실패 재시도 방식

## Record 재생성 기술

- 기존 Context 행의 `record_id`를 변경할지, 연결 테이블로 분리할지
- Record 재생성 중 사용자 조회를 막는 상태값
- 기존 Record 삭제와 새 Record 활성화 순서
- 임베딩 실패 시 롤백 또는 보상 트랜잭션
- `previous_record_id` 등 버전 계보 필드 도입 여부

## Feed

- 추천 정렬 알고리즘
- 개인화 초기 버전
- 노출 중복 방지
- 페이지네이션 방식

## UX

- Record 삭제와 Collection 연쇄 삭제 확인창 문구
- 마지막 Context 삭제 시 다중 Collection 영향 표시 방식
- 여러 Place를 한 번에 가져올 때 Context 입력 UX
- 자동 발행 약관 문구와 별도 체크박스 여부
