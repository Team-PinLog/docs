# PinLog 공식 용어사전

| 한글 | 영문 | 정의 | 아닌 것 |
|---|---|---|---|
| 사용자 | User | 서비스 계정의 주체 | 공개 프로필 |
| 장소 | Place | 카카오맵에서 유래한 실제 지점의 공용 정보 | 사용자 소유 기록 |
| 레코드 | Record | 한 User와 한 Place의 연결 단위. 활성 Context를 1개 이상 가짐 | 리뷰 텍스트 |
| 맥락 | Context | Record 안의 독립 서술 단위 | 댓글, 단순 키워드 |
| 키워드 | Keyword | AI가 프리셋에서 매핑한 비식별화 검색·추천 표현 | 사용자 입력 태그 |
| 컬렉션 | Collection | Record를 1개 이상 묶은 사용자 정의 공개 그룹 | 폴더, Place 직접 묶음 |
| 컬렉션 레코드 | CollectionRecord | Collection과 Record의 N:M 연결 관계 | Record 원본 |
| 선반 | Shelf | 한 User가 발행한 Collection 목록 | 개인 Record 목록 |
| 라이브러리 | Library | 내 Shelf와 팔로우한 Shelf를 관리하는 개인 공간 | 공개 프로필 |
| 팔로우 | Follow | 특정 Shelf를 Library에서 지속 확인하는 관계 | User 신원 팔로우 |
| 피드 | Feed | 발행된 Collection을 익명 추천하는 영역 | 검색 결과 |
| AI 자연어 검색 | AI Record Search | 내 Context를 의미 기준으로 검색해 Record로 반환하는 기능 | 카카오맵 장소 검색 |
| 장소 검색 | Place Search | 카카오맵 API에서 실제 지점을 조회하는 기능 | 개인 Record 검색 |
| 소셜 계정 | SocialAccount | User에 연결된 외부 로그인 수단 | 서비스 User 자체 |
| 임베딩 | Embedding | AI 자연어 검색을 위한 벡터 표현 | 공개 Keyword |
| 키워드 프리셋 | Keyword Preset | AI가 선택할 수 있는 사전 정의 Keyword 목록 | AI가 자유 생성한 단어 |
| 임베딩 프로필 | Embedding Profile | 모델·차원·거리 기준을 식별하는 설정값 | 모델 이름 단독 |
| AI 상태 | AI State | Context별 AI 처리 진행 상태 | Record 공개 상태 |
| 맥락 버전 | Context Version | Context 본문 수정 시에만 증가하는 값 | 낙관적 락 버전 |
| 피드 이벤트 | Feed Event | Feed 추천 결과에 대한 노출·클릭·저장 기록 | 사용자 활동 로그 전반 |
| 노출 | Impression | Feed 응답으로 Collection이 전달된 사건 | 화면에 실제로 보인 사건 |

## Keyword 공개 등급

| 등급 | 의미 |
|---|---|
| PUBLIC | 본인과 타인에게 모두 제공하며 추천 계산에 사용 |
| PRIVATE_ONLY | 본인에게만 제공하며 본인 개인화 계산에만 사용 |
| BLOCKED | 본인과 타인 모두에게 제공하지 않으며 모든 계산에서 제외 |

등급별 상세 계산 범위는 [AI 설계](05_AI_설계.md)를 따릅니다.
