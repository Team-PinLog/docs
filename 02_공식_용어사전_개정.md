# PinLog 공식 용어사전 — 개정본

| 한글 | 영문 | 정의 | 아닌 것 |
|---|---|---|---|
| 사용자 | User | 서비스 계정의 주체 | 공개 프로필이 아님 |
| 장소 | Place | 카카오맵에서 유래한 실제 지점의 공용 정보 | 사용자 기록이 아님, 사용자 소유가 아님 |
| 레코드 | Record | 한 User와 한 Place의 연결 단위. 활성 Context를 1개 이상 가진다. User당 Place별 활성 Record는 최대 1개다. | 리뷰 텍스트 자체가 아님 |
| 맥락 | Context | Record 안에 포함된 독립 서술 단위. 동행자, 목적, 분위기, 경험, 저장 이유 등을 작성한다. | 댓글이 아님, 단순 키워드가 아님 |
| 키워드 | Keyword | AI가 Place와 Context를 분석하여 사전 정의 목록에서 매핑한 비식별화 검색·추천 표현 | 사용자 입력 태그가 아님 |
| 컬렉션 | Collection | Record를 1개 이상 묶은 사용자 정의 그룹. 생성 즉시 자동 발행된다. | 폴더가 아님, Place나 Context를 직접 담지 않음 |
| 컬렉션 레코드 | CollectionRecord | Collection과 Record의 N:M 연결 관계 | Record 원본이 아님 |
| 선반 | Shelf | 한 User가 발행한 Collection 목록. 다른 사용자는 신원을 모른 채 Shelf를 팔로우한다. | 개인 Record 목록이 아님 |
| 라이브러리 | Library | 내 Shelf와 팔로우한 Shelf를 함께 관리하는 개인 공간 | 공개 프로필이 아님 |
| 팔로우 | Follow | 특정 Shelf의 Collection을 Library에서 지속적으로 확인하는 관계 | User 신원을 팔로우하는 관계가 아님 |
| 피드 | Feed | 발행된 Collection을 익명·개인화 추천하는 영역 | 검색 결과 화면이 아님 |
| AI 자연어 검색 | AI Record Search | 내 Place, Context, Keyword를 검색하고 Record 단위로 반환하는 기능 | 카카오맵 장소 검색이 아님 |
| 장소 검색 | Place Search | 카카오맵 API를 대상으로 실제 지점을 조회하는 기능 | 개인 Record 자연어 검색이 아님 |

## 폐기 용어

### Memory

기존 문서에서 개인 기록을 뜻하던 `Memory`는 폐기한다. 모든 문서, API, DB, UI에서는 `Record`와 `Context`로 분리하여 사용한다.

### 장소 모음집

공식 용어는 `Collection`으로 통일한다.

### 익명 책장

공식 용어는 `Shelf`로 통일한다.

### 발견 피드

공식 용어는 `Feed`로 통일한다.

### 개인 Archive

공식 용어는 `Library`로 통일한다.
